# Rafhin Visuals — Portfolio Site

Personal portfolio website for **Rafhin Visuals**, a video and photo production studio based in Canada.

## Pages

- **Home** (`index.html`) — Hero, 3D rotating carousel, expertise, brands, and CTA
- **Work** (`work.html`) — Filterable project grid
- **Services** (`services.html`) — Service offerings and collaboration CTA
- **Contact** (`contact.html`) — Contact info and booking

## Stack

- Vanilla HTML/CSS — single-file pages, all styles inline
- [Tailwind CSS](https://tailwindcss.com/) via CDN
- [GSAP 3.12.5](https://greensock.com/gsap/) for nav overlay animations
- Google Fonts: Archivo, IvyPresto Display, Azeret Mono

## Local Development

```bash
node serve.mjs
```

Opens at `http://localhost:3000`.
